# 🚀 StudyFlow — AI-Powered Study Planner & Practice Platform

StudyFlow is an intelligent web application designed to help students prepare smarter for board exams.
It leverages **AI-driven planning + real-time practice + performance tracking** to create a complete study ecosystem.

---

## 🤖 AI-Powered Intelligence

At the core of StudyFlow is an AI engine powered by **Google Gemini API**, which:

* Generates **personalized study plans** based on:

  * Class, subjects, chapters
  * Available study time
  * Preferred start time
* Balances study sessions with breaks for **maximum focus**
* Produces structured, optimized schedules in real-time

> Instead of guessing what to study, students get a **clear, AI-optimized roadmap instantly.**
